Sobre o Projeto

O projeto Smart Campus SENAI 2025 nasceu de um desafio real observado em nosso dia a dia: como podemos utilizar a tecnologia para otimizar os recursos da nossa escola e tornar o ambiente mais interativo? Com base nos conhecimentos adquiridos no curso Técnico em Desenvolvimento de Sistemas, nosso objetivo é criar uma solução prática que integre sensores de Internet das Coisas (IoT) para coletar dados em tempo real e uma plataforma web para visualizar e analisar essas informações, contribuindo para um campus mais eficiente e sustentável.

Para gerenciar o desenvolvimento, adotamos uma metodologia ágil, organizando nossas atividades em um quadro Kanban no Trello e documentando cada etapa. A solução envolve a criação de uma aplicação web completa, a programação de microcontroladores para os sensores de IoT e a estruturação de um banco de dados. Este projeto é a nossa principal Situação de Aprendizagem, onde aplicamos diretamente as competências técnicas e socioemocionais desenvolvidas em sala de aula para resolver um problema concreto.

Monitoramento Inteligente de Sala

Monitora temperatura, umidade e ocupação, enviando dados para o dashboard para otimizar o uso de ar-condicionado.

Controle de Iluminação Automatizado

Acende as luzes apenas quando há pouca luz natural e presença de pessoas, reduzindo o consumo de energia.

Alerta de Segurança para Portas

Detecta aberturas de portas ou janelas fora do horário e envia um alerta, aumentando a segurança dos laboratórios.

Controle de Acesso com NFC/RFID

Libera o acesso a laboratórios apenas para usuários autorizados via cartão, registrando entradas e saídas.

Detector de Vazamento de Água

Detecta vazamentos ou chuva, enviando alertas para a manutenção e evitando danos e desperdício.

Controle Logístico de Carrinhos

Registra a retirada e devolução de carrinhos de notebook, identificando o responsável via TAG NFC/RFID.

Sistema de Empréstimo da AAPM

Autoatendimento para empréstimo de itens (violão, jogos), onde o aluno se identifica e registra o item com RFID.

Ativação de Bancadas por Voz

Permite ligar e desligar bancadas de laboratórios através de comandos de voz, modernizando e aumentando a segurança.

Secretaria Virtual por Voz

Totem interativo que responde por voz às principais dúvidas de alunos e visitantes, otimizando o atendimento.

Monitoramento de Ocupação de Labs

Sistema que mostra qual professor está usando a sala via crachá NFC, otimizando a distribuição dos ambientes.
